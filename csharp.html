about this course

C# is one of the most popular programming languages and can be used for a variety of things, including mobile applications, game development, and enterprise software. Knowing C# opens a great deal of doors for you as a developer.In this course, you’ll be exposed to fundamental programming concepts using C# and start writing programs right away. You’ll build several projects to help you practice and test your knowledge using quizzes.
